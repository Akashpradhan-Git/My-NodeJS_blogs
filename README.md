I this Prject i build a full stack NodejS App where user can add new Article, Edit Them and delete the same article.

##### Technology used

- NodeJS
- Express JS
- MongoDB
- Mongo Atlas
- Ejs(Template Engine)
- ✨Vs Code✨

## Features

- Add New Article
- Edit Article
- Delete Article
- View Perticular Blog

## Rest API Path

- [/] - Serve Home Page
- [/new] - For Creating New Article
- [/edit/:id] - for edit Perticular blog
- [/delete/:id] - Delete the blog.

## Installation

```sh
cd dillinger
npm i
npm i express
npm i nodemon
npm i mongooes
```

#### For production environments

```sh
npm install --production
npm start
```

#### Development

```sh
npm install --production
npm run dev
```
